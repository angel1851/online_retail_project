# Dataset Source

The dataset used in this project is the **Online Retail Dataset**.

Due to GitHub file size limitations, the dataset is not included in this repository.

You can download the dataset here:
https://drive.google.com/file/d/1P90OcUywnDlikNyZo-jy5CnvrmdKmGvd/view?usp=drive_link

After downloading, place the dataset inside the `data/` folder before running the SQL scripts.
